Welcome

My name is Jaakko Lipsanen. I am an 20-year-old game developer from Helsinki, Finland. Most of my completed projects can be found on this site.

I have programmed and developed games for three years. During that time, I have released several games for Windows Phone with over 1,000,000 downloads.

I consider myself to be proficient in C# and Java and intermediate in C++. I also know the basics of JavaScript, Clojure, Python as well as HTML and CSS (this page proves that!).

I have developed my games mainly using C# and XNA, but recently I have switched to using Unity. I have also made some projects using C++ and OpenGL/SFML.

I graduated from high school in the spring of 2014 and I am planning to begin studying Computer Science at the University of Helsinki in the fall of 2016.
