My Games

Final Fight Z

Final Fight Z is a top-down twin-stick shooter where you try to survive from hordes of zombies by using various different weapons. The game features global leaderboards, achievements and a great pixel-art graphics.

Final Fight Z was made in the fall of 2013 over a period of 3 months using C# and XNA. It is available for free in the Windows Phone Store.

Doodle Run

Doodle Run is an addictive endless runner with great doodle graphics, random generated maps and global leaderboards.

I developed Doodle Run in the late 2012 in one month using C# and XNA. The game was a suprise success with over 700,000 downloads. It is available for free in the Windows Phone Store.

boxStrike

In boxStrike, your goal is to evade other boxes and survive as long as possible. It features four unique graphical themes and global leaderboards.

boxStrike was the first game I published. The development took me about one month and I used C# and XNA. It is available for free in the Windows Phone Store.

Installation ZA-3

Installation ZA-3 tells a story of a scientist named Jack, who is a part of a group exploring a far away planet. One day, Jack finds himself to be in the middle of an alien invasion!

Installation ZA-3 was developed in 72 hours for Ludum Dare #29 using Unity. It was my first game developed using Unity and also the first game featuring an actual story and conversations.
