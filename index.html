Games

    Final Fight Z

    Final Fight Z is a top-down twin-stick shooter where you try to survive from hordes of zombies by using various different weapons. The game features global leaderboards, achievements and a great pixel-art graphics.

    Final Fight Z was made in the fall of 2014 in a period of 3 months using C# and XNA. It is available for free in the Windows Phone Store.

    Doodle Run

    Doodle Run is an addictive endless runner with great doodle graphics! Doodle Run has random generated maps and global leaderboards.

    I developed Doodle Run in the late 2012 in two months using C# and XNA. The game was a suprise success with over 800,000 downloads. It is available for free in the Windows Phone Store.

    boxStrike

    In boxStrike, your goal is to evade other boxes and survive as long as possible. It features four unique graphical themes and global leaderboards.

    boxStrike was the first game I published. The development took me about one month and I used C# and XNA. It is available for free in the Windows Phone Store.
